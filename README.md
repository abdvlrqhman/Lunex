## 🔧 Troubleshooting

### 🛡️ **Windows Defender Fix** (Most Common Issue)

<div align="center">

**⚠️ Important:** Windows Defender may block Lunex. Follow these steps to fix:

</div>

<details>
<summary><b>🛡️ How to Whitelist Lunex in Windows Defender</b></summary>

<div align="center">

| Step | Action |
|:----:|--------|
| **1** | Open **Windows Security** |
| **2** | Go to **Virus & Threat Protection** |
| **3** | Click **Manage Settings** under 'Virus & Threat Protection Settings' |
| **4** | Scroll to **Exclusions** and click **'Add or remove exclusions'** |
| **5** | Click **Add an exclusion** → **Select Folder** |
| **6** | Choose the **Lunex installation directory** |

</div>

**✅ This resolves 90% of installation and connection issues!**

</details>

### 🔍 Other Common Issues

<details>
<summary><b>🚫 Connection Problems</b></summary>

**🔌 Connection Failed**
- ✅ Check internet connectivity
- ✅ Restart Lunex application  
- ✅ Disable other VPN software
- ✅ Run as Administrator
- ✅ Contact support if issues persist

**🎙️ Voice Chat Still Not Working**
- ✅ Confirm Lunex shows "Connected"
- ✅ Check in-game voice settings
- ✅ Restart game completely
- ✅ Try disconnecting and reconnecting Lunex

</details>

<details>
<summary><b>🎮 Game-Specific Issues</b></summary>

**🚀 Game Won't Launch**
- ✅ Verify game is in supported list
- ✅ Close all background VPN software
- ✅ Run both Lunex and game as Administrator
- ✅ Check firewall settings

**⚡ Performance Issues**
- ✅ Close unnecessary background applications
- ✅ Use wired internet connection
- ✅ Select optimal server location
- ✅ Update Lunex to latest version

</details>
